# README — service-cursos-linkding-GPT

## Descrição

Serviço para integração entre cursos online e o Linkding, utilizando GPT para automação de tarefas, sugestões e organização de links.

## Funcionalidades

- Automatização de cadastro de cursos
- Sugestão de links e recursos
- Integração com Linkding

## Como rodar

```bash
npm install
node init.js
```
