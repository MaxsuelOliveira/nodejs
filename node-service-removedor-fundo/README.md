# BG Remover Pro

Projeto front-end puro para remoção automática de fundo com foco em chroma key verde e azul.

## Recursos

- HTML + CSS + JS separados
- Drag and drop
- Comparação deslizante antes/depois
- Detecção automática de fundo verde ou azul
- Processamento com HSV
- Despill para bordas coloridas
- Limpeza de ruído com blur + filtro mediano simplificado
- Exportação em PNG transparente
- Interface com estilo SaaS

## Como executar

Basta abrir o arquivo `index.html` no navegador.

## Estrutura

- `index.html`
- `styles.css`
- `app.js`
- `README.md`

## Observações

Este projeto funciona melhor em imagens com fundo relativamente uniforme.

Em cenários com cabelo complexo, sombras fortes, transparências e fundo não uniforme, o resultado pode exigir ajuste manual nos controles.
